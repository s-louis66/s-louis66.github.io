Main Profile Page

    The header of my current portfolio web page that is already live and is currently missing several pages for the About Me section (that I intend to add later) so I made it into a web page for now. It is currently only responsive for small devices, laptops and desktop computers and includes several design and interactive features that captivates the users attention. This is a hand-coded website made with HTML(HTML5,Canvas), CSS(CSS3), JavaScript(jQuery), Bootstrap and was started from scratch when I first started learning web development and has been an on-going project since.

    A responsive, Bootstrap nav bar that includes hover effects for the nav items, a smooth, navigation transition on each nav item click, and also, for desktop view, a change of height for the nav bar that fires when you scroll down. The banner is a 2-column split that hides the left side on small devices but has a parallax feature on desktop view. The right side is just a quote from one of my favorite books. I intend to upgrade this in the future to a quote that changes either daily or weekly with javaScript or jQuery.

    A simple, 2-column split with a Bootstrap card on the left side and 2 contact buttons. A right side that fires when scrolled into view with fade-in items that are timed via CSS.

    A section header that fires when scrolled into view.
